# Smart Building Energy Consumption Optimization

## Objectif
Ce projet simule un cas de monitoring énergétique de bâtiment intelligent visant à exploiter des données de consommation électrique pour identifier les périodes de pointe, détecter les anomalies de charge et proposer des pistes d’optimisation énergétique.
